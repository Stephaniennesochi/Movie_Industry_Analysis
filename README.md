# Movie_Industry_Analysis (Interactive Dashboard Creation using Power BI)
## Project Objective
To analyze the financial performance, ratings, and genre trends of movies using the TMDB Movies Dataset and develop an interactive Power BI dashboard that provides insights into profitability, audience reception, and industry patterns for data-driven decision-making.

## Dataset Used
<a href = "https://github.com/Stephaniennesochi/Movie_Industry_Analysis/blob/master/assets/datasets/TMDB_movie_dataset_v11.csv"> Movie Industry Data </a>

## Questions(KPIs)
- What is the total revenue generated by all movies?
- What is the total production budget?
- What is the total profit generated?
- What is the overall Return on Investment (ROI)?
- How many movies are included in the dataset?
- What is the average movie rating?
- Which movies are the top 10 highest rated?
- Which genre generates the highest revenue?
- Which genre is the most profitable?
- Which genre has the highest ROI?
- How many movies belong to each genre?
- How has movie revenue changed over time?
- Has profitability improved over the years?
- Is there a relationship between budget and revenue?
- Does a higher rating lead to higher revenue
