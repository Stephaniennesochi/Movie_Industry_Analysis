# Movie_Industry_Analysis (Interactive Dashboard Creation using Power BI)
## Project Objective
To analyze the financial performance, ratings, and genre trends of movies using the TMDB Movies Dataset and develop an interactive Power BI dashboard that provides insights into profitability, audience reception, and industry patterns for data-driven decision-making.
